# leetcode-practice

LeetCode solutions in Python with unit tests, complexity analysis, and clean code structure for interview preparation.

This repo also includes a Python 3 Snake game in the `snake_game/` folder.

## Run

```bash
cd /Users/jarrel/Documents/Playground
python3 -m snake_game.main
```

## Test

```bash
cd /Users/jarrel/Documents/Playground
python3 -m unittest discover -s tests
```

## Manual checks

- Start with arrow keys or `WASD`
- Confirm the snake moves on the grid
- Eat food and confirm the score increases
- Hit a wall or the snake body and confirm game over
- Press `Space` or `Pause` to pause and resume
- Press `Restart` to reset the game
